# AI Study Hub – Frontend Prototype

Prototype giao diện dùng để trình bày và review. Project sử dụng hoàn toàn mock data, không gọi backend.

## Chạy project

```bash
pnpm install
pnpm dev
```

Sau đó mở địa chỉ Vite hiển thị trong terminal (thường là `http://localhost:5173`).

## Luồng demo

1. Đăng nhập bằng dữ liệu có sẵn hoặc nút Google.
2. Xem Dashboard và tài liệu gần đây.
3. Chuyển sang Tài liệu, đổi chế độ lưới/danh sách và thử upload giả.
4. Mở Trợ lý AI, nhập câu hỏi và nhận câu trả lời mẫu.
5. Xem Nhóm học tập, Dung lượng và Hồ sơ.

Không cần cấu hình `.env`, database hoặc backend.
